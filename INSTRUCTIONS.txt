Instructions on how to run this project locally

There are two options:

Option 1: Use VS Code Live Server

    1. Open the project folder in Visual Studio Code
    2. Install the extension “Live Server”
    3. Right-click the file index.html
    4. Choose “Open with Live Server”
    5. The visualization will open in your browser.

Option 2: Use Python’s Built-In Web Server

    1. Open a terminal / command prompt
    2. Navigate to the project folder:
    3. cd path/to/the/project
    4. Type in:

`python -m http.server`

    5. Open the this link in your browser:

http://localhost:8000
